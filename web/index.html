Features

  • ✨ Predicate suggestions active
    Adds a Suggest with AI button to the predicate editor. Asks the primary model to propose predicates from the cron's prompt + recent run history. Results are appended to the form for your review — nothing is saved without an explicit Save.
  • ⏳ Healthcheck suggestions active
    AI-proposed pre-retry healthchecks (e.g. "vLLM is responding on :8100"). Click the healthchecks badge on any cron to manage them and use the Suggest with AI button. Enforcement during retries (skipping a retry when a healthcheck fails) lands in a follow-up release; for now this manages the rules.
  • 🔢 Smart retry tuning planned
    Observe each cron's retry success rate and suggest a per-cron max_retries value backed by the data.
  • 💾 Failure-mode explanations planned
    When an alert fires, include an AI-generated one-paragraph diagnosis of the likely root cause based on the failure pattern + the cron's prompt.
